# Second Look

Second Look is a privacy-preserving, on-device mammogram analysis prototype. This branch (`integration/data-pipeline`) integrates the Phase 1 pipeline: a binary label mapper (CBIS-DDSM, RSNA, VinDr → `WORTH_SECOND_LOOK` / `NOT_WORTH_SECOND_LOOK`) with UX tier helpers, a preprocessor (CLAHE, breast masking, pectoral removal, orientation, 224×224 float32 output), an input `quality_check` gate, a stratified 70/15/15 splitter, and a MobileNetV2 baseline classifier with a sigmoid head, binary-crossentropy training, and sensitivity-floor evaluation. Tests cover label round-trips and a fixture-gated preprocessing smoke test. Not yet merged: augmentation module. Not yet written: GCS retriever, manifest builder, `build_dataset` CLI. See `second_look_work_breakdown_structure.docx`.
